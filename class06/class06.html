We have our working snippet of code! This is now going to be the body of our function.

All function in R have at least 3 things:

  • A name (we pick that)
  • input arguments
  • body (the code that does the work)
grade <- function(x) {
  #mask NA to zero
  x[is.na(x)] <- 0
  # drop the lowest value and get mean
  mean( x[-which.min(x)]) }
